# 個人記帳系統💸💸💸
##  專案簡介 
一個實用的個人記帳系統，採用 CSV 純文字檔案儲存資料，無需安裝資料庫即可直接執行。內建圖表分析、音效互動、動態背景等功能，是一個非常強大的工具!!!

##  功能特色
* 記帳管理：新增 / 編輯 / 刪除收支紀錄，DataGridView 即時顯示
* 篩選查詢：年份、月份篩選 + 關鍵字即時搜尋（描述 / 類別）
* 統計摘要：即時計算總收入、總支出、結餘
* 圖表分析：支出類別佔比圓餅圖、月份收支對比長條圖
* 視覺客製：類別專屬背景色、動態漸變背景、單色背景切換
* 音效互動：背景音樂播放、操作點擊音效
* 資料匯出入：CSV 匯出（相容 Excel）、CSV 匯入

##  使用方式
* 檔案：匯入或匯出CSV檔案
* 功能
  * 新增紀錄：新增記帳紀錄
  * 查看圖表：顯示支出類別佔比圓餅圖、月份收支對比長條圖
* 背景：可以選擇單色，或使用漸層背景
* 音樂：開啟音檔當作BGM，或是繼續 / 暫停音樂
* 說明：開啟關於頁面
* 篩選：選擇年月份，點擊篩選找出目標資料
* 尋找：依描述 / 類別關鍵字即時搜尋
